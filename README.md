Objective:
The objective of this project is to develop a simple and user-friendly To-Do List application that helps users organize and manage their daily tasks efficiently. The application allows users to add, mark as completed, and delete tasks while storing the data locally in the browser using Local Storage.
Tools and Techniques:
•HTML5 – Creates the structure of the web page.
•CSS3 – Designs the user interface with responsive layout, gradients, animations, and hover effects.
•JavaScript (ES6) – Implements the application's functionality and dynamic behavior.
•Local Storage API – Saves tasks in the browser so they remain available after refreshing the page.
•DOM Manipulation – Dynamically creates, updates, and deletes task elements.
Features:
•Add new tasks.
•Delete unwanted tasks.
•Mark tasks as completed.
•Automatically saves tasks using Local Storage.
•Tasks remain available after page refresh.
•Responsive and modern user interface.
•Beautiful gradient •background and interactive buttons.
•Displays an empty-state message when no tasks are available.
•Supports adding tasks by pressing the Enter key.
Smooth hover effects and transitions.
Output:
The application provides a clean and attractive To-Do List interface where users can manage their daily tasks. Users can add new tasks, mark them as completed, delete completed or unwanted tasks, and their task list is automatically saved in the browser, ensuring data is retained even after the page is refreshed.
